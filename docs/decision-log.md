# Trace View Core Decision Log

## 2026-03-26

- 제품명은 `Trace View Core`로 확정
- 기획 및 설계 단계 완료 전까지 구현은 시작하지 않음
- 기획 및 설계는 멀티 에이전트 방식으로 진행
- 메인 에이전트가 전체 조율을 담당
- 문서와 결과물은 별도 Validation Agent가 검증
- 진행 현황은 별도 Status Agent 기준으로 관리
- 화면 정의는 MVP에서 라우트 기반 페이지 단위로 설정
- 비즈니스 로직은 서비스 또는 유스케이스 계층 중심으로 제한
- 영향도 기능은 MVP에서 연결 노드 탐색 중심으로 제한
- 자동 생성 비즈니스 요약은 Draft 승인 모델을 따름
